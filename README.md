# Clínica Vida Plena - Site Institucional

## Descrição
Este é um site institucional para a **Clínica Vida Plena**, desenvolvido como modelo ou template para apresentação online de serviços de saúde.  
O site apresenta informações sobre a clínica, horários de atendimento, contato e equipe, com um layout simples e responsivo.

---

## Estrutura do Projeto

```text
/ (raiz do projeto)
│
├── index.html           # Página principal
├── /src
│   ├── /html
│   │   ├── sobre.html       # Página "Sobre a Clínica"
│   │   ├── horarios.html    # Página de horários
│   │   └── contato.html     # Página de contato com formulário
│   └── /css
│       ├── style.css        # Estilos principais
│       └── responsive.css   # Estilos para responsividade (opcional)
└── README.md
```

---

## Funcionalidades
- Menu lateral com links para todas as páginas do site.
- Página principal com imagem de destaque e texto de apresentação.
- Página "Sobre a Clínica" com missão, valores e estrutura.
- Página de horários com tabelas de atendimento.
- Página de contato com informações de telefone, WhatsApp, e-mail e formulário de mensagem.
- Layout flexível e centralizado, usando CSS com cores personalizadas.
- Responsivo para diferentes tamanhos de tela (com ajustes no CSS).

---

## Como Usar
1. Baixe ou clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Navegue pelo menu para acessar as outras páginas.
4. (Opcional) Customize imagens, cores e textos no CSS e HTML.

---

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (opcional)

---

## Observações
- Todas as imagens utilizadas são provenientes de URLs externas (Unsplash).
- Formulário de contato é apenas ilustrativo e não envia mensagens de fato.
- Recomendado abrir o projeto em um servidor local para testes completos de navegação.
